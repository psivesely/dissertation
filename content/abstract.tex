\begin{spacing}{2}
Succinct Non-interactive ARguments of Knowledge (SNARKs) produce small proofs that can be verified quickly while attesting to the correctness of arbitrary computations. First deployed in 2016, SNARKs have enabled private transactions, decentralized storage, anonymous credentials (e.g., private age verification), proofs about network sessions (e.g., email and TLS session contents), and more. This dissertation advances SNARKs in two ways: (1) it develops arithmetization techniques and polynomial interactive oracle proofs (PIOPs) for mixed- and small-domain computations, enabling constraints to be written over their natural domains and reducing non-native overhead; and (2) it gives efficient post-quantum (hash- and lattice-based) polynomial commitment schemes (PCSs) for these specialized domains.

\emph{Zinc+} is a framework for proving computations specified across multiple rings and ideals simultaneously. We define the Universal Constraint System (UCS), capturing constraints over $\ZZ,\QQ,$ finite fields, and polynomial rings together with ideal-membership predicates. The Zinc+ compiler lifts finite-field polynomial PIOPs into PIOPs for UCS. Our novel hash-based PCS, Zip+, supports these mixed-domain coefficients. \emph{Microlotus} presents a PIOP for cyclotomic ring arithmetic via slot-wise sumcheck over the base field, avoiding the quotienting overhead of Galois-ring projection approaches. We implement a PCS built on Basefold, random foldable codes, and an odd-prime field tower enabling zero-overhead extension field embedding, with performance competitive with state-of-the-art schemes over more structured fields. Finally, \emph{Orbweaver} presents a lattice-based functional commitment for cyclotomic rings and the integers, the first to achieve a quasilinear prover with a log-time verifier. We derive a PCS from Orbweaver and show it supports public aggregation.
\end{spacing}
